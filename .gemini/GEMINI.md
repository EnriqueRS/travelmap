## Project Context
Travelmap is a web application that allows users to track their travels and share them with others. It uses SvelteKit for the frontend and NestJS for the backend. The database is PostgreSQL with Prisma.

## Standards
- Development allways for mobile first
- All comments and documentation must be in English
- README.md must be in English and Spanish
- Allways update README.md with the changes made

## Constraints
- No direct DB queries in components
